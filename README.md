# COC API
Clash of clan website for getting detailed list of FWA(Farming War Alliance) clans.


# Sources:
    - https://developer.clashofclans.com/#/documentation
    - https://www.clashofstats.com/
    - http://fwastats.com/


## Usage
```
npm install
npm start

Go to localhost:3001
```
